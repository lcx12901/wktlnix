# TOOLS.md — 工具选择规则

## 优先层级（从廉到贵）

### Level 1: 本地·免费·即时（首选）
- **read** → 读任务需求、已有研究报告、本地文件、共享知识
- **exec** → 运行脚本处理结构化数据
- **memory_recall** → 搜索历史记忆（只读，跨 session 查之前的调研结论/关键发现）

### Level 2: 网络·轻量（次选）
- **web_search** → 搜索主题相关文章、文档、竞品信息
- **web_fetch** → 抓取搜索结果中的具体页面内容

### Level 3: LLM 密集型（最小化）
- **sessions_spawn** → 仅独立子任务

## 错误恢复
### 可恢复（自动处理）
- HTTP 429/503 → 等待 10s 重试，3 次后跳过此源
- 页面抓取失败 → 换搜索关键词或来源，继续
- 搜索结果不足 → 换搜索词/语言/地区重试 1 次

### 不可恢复 → 停止 + 报告 Nova
- 搜索结果全部失败（网络不可用）
- 所有来源返回 403/401
- 信息严重冲突需 Nova 决策下一步

## 硬边界
- 不编造引用（来源不可信时标注"不可信"）
- 不照搬不可信源原文（总结 + 标注可信度）
- 不越过任务范围扩大调研（请示 Nova）
- 不在输出中包含未脱敏的个人信息
