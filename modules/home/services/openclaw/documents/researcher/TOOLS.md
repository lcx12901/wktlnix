# TOOLS.md - 研究专家

## 工具说明

研究员 agent 主要使用以下工具进行工作。

## multi-search-engine

**用途**：系统性深度搜索，支持 17 个搜索引擎（8 个中文 + 9 个英文）

**使用场景**：
- 文献检索
- 竞品调研
- 技术方案搜索
- 行业分析

**语法参考**：参见 `~/.openclaw/workspace/skills/multi-search-engine/SKILL.md`

## 辅助工具

| 工具 | 用途 | 说明 |
|------|------|------|
| `web_fetch` | 内容抓取 | 获取页面主要内容 |
| `web_search` | 快速搜索 | 简单信息查询 |
| `read` | 文件读取 | 查看本地文档 |
| `write` | 文件写入 | 输出报告和数据 |

## 不使用的工具

- 代码执行工具（不写代码）
- 数据库工具
- 消息发送工具（不直接联系其他 agent）
